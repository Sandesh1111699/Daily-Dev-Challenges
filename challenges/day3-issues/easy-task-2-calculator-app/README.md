# Day 3 - EASY Task 2: Build Calculator App

A fully functional calculator application with a clean, modern interface and keyboard support.

## Features

- ✅ Basic arithmetic operations (+, -, ×, ÷)
- ✅ Decimal number support
- ✅ Clear (AC) button to reset calculator
- ✅ Delete (DEL) button to remove last digit
- ✅ Display shows current and previous operand
- ✅ Keyboard support for all operations
- ✅ Prevents multiple decimals in single number
- ✅ Error prevention for incomplete operations
- ✅ Smooth transitions and hover effects
- ✅ Responsive design
- ✅ Dark theme with gradient buttons

## Technologies Used

- HTML5 (semantic markup)
- CSS3 (grid layout, flexbox, gradients, animations)
- Vanilla JavaScript (ES6 classes, event listeners)

## How to Use

### Mouse/Touch
1. Click number buttons to enter numbers
2. Click an operator (+, -, ×, ÷) to choose operation
3. Click another number to enter the second operand
4. Click = to see the result
5. Use AC to clear and start over
6. Use DEL to remove the last digit

### Keyboard
- **Numbers:** 0-9, .
- **Operators:** +, -, *, /
- **Calculate:** Enter or =
- **Delete:** Backspace
- **Clear:** Escape

## Operations Supported

| Operator | Key | Symbol |
