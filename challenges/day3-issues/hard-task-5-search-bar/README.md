# Day 3 - HARD Task 5: Create Search Bar

A comprehensive search interface with advanced filtering, autocomplete suggestions, and an advanced search modal.

## Features

- ✅ Real-time search with instant results
- ✅ Autocomplete suggestions dropdown
- ✅ Multiple filter options (category, price range, sort)
- ✅ Clear button for quick reset
- ✅ Advanced search modal with complex queries
- ✅ Result count display
- ✅ Different product categories
- ✅ Price range slider
- ✅ Multiple sorting options
- ✅ Responsive design
- ✅ Keyboard shortcuts (Enter, Escape)
- ✅ Smooth animations

## Technologies Used

- HTML5 (semantic forms)
- CSS3 (flexbox, grid, animations, modals)
- Vanilla JavaScript (search algorithms, filtering, sorting)

## Features Breakdown

### Basic Search
- Type to search products by name or category
- Real-time results update
- Autocomplete suggestions appear as you type
- Click suggestions to auto-fill

### Filtering
- **Category:** Filter by product type (Electronics, Clothing, Books, Food, Sports)
- **Price Range:** Slider to set maximum price
- **Sort:** Multiple sorting options (Relevance, Price Low/High, Newest, Most Popular)

### Advanced Search Modal
- **Exact Match:** Search for exact phrase
- **All Words:** Search products containing all specified words
- **Any Word:** Search products containing any of the words
- **Exclude:** Remove products with specific words

## How to Use

1. **Search:** Type in the search box to find products
2. **Suggestions:** Click a suggestion to select it
3. **Filters:** Use dropdowns and sliders to refine results
4. **Advanced:** Click "Advanced Search" for complex queries
5. **Results:** View matching products with category and price

## Keyboard Shortcuts

- **Enter:** Perform search
- **Escape:** Close modal or suggestions
