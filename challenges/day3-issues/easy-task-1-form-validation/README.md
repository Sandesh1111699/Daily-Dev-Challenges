# Day 3 - EASY Task 1: Create Form Validation

A comprehensive form validation application with real-time feedback and error handling.

## Features

- ✅ Real-time field validation on blur and input
- ✅ Custom validation rules for each field
- ✅ Email format validation
- ✅ Phone number validation (10-digit, with hyphens or spaces)
- ✅ Password strength requirements (8+ chars, uppercase, numbers)
- ✅ Password confirmation matching
- ✅ URL/Website validation
- ✅ Terms acceptance checkbox
- ✅ Visual feedback (green for valid, red for invalid)
- ✅ Error messages for each field
- ✅ Submit and reset functionality
- ✅ Responsive design for all devices

## Technologies Used

- HTML5 (semantic markup)
- CSS3 (gradient backgrounds, flexbox, responsive design)
- Vanilla JavaScript (DOM manipulation, regex validation)

## How to Use

1. Open `index.html` in a web browser
2. Fill in the form fields
3. Fields validate in real-time as you interact with them
4. View error messages for invalid inputs
5. Complete all required fields correctly
6. Click "Submit" to submit the form
7. Use "Clear" button to reset all fields

## Validation Rules

| Field | Requirements |
|-------|--------------|
| Full Name | Minimum 3 characters |
| Email | Valid email format |
| Phone | 10-digit number (XXX-XXX-XXXX or variations) |
| Password | 8+ characters, uppercase letter, number |
| Confirm Password | Must match password field |
| Website | Valid URL format (http/https) or empty |
| Terms | Must be checked |
