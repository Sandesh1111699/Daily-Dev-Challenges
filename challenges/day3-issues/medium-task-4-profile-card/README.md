# Day 3 - MEDIUM Task 4: Build Profile Card UI

A beautiful and interactive profile card component showcase with hover effects and smooth animations.

## Features

- ✅ Responsive grid layout (1-4 columns based on screen size)
- ✅ Profile cards with avatar images
- ✅ Professional information display (name, title, bio)
- ✅ Statistics section showing achievements
- ✅ Social media links with hover effects
- ✅ Follow and Message buttons with interactive states
- ✅ Smooth hover animations with elevation
- ✅ Gradient backgrounds
- ✅ Image hover zoom effect
- ✅ Fully responsive design
- ✅ Touch-friendly on mobile

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (grid, flexbox, gradients, animations, transitions)
- Vanilla JavaScript (event listeners, state management)

## Profile Information

Each card displays:
- **Avatar:** Randomly generated profile picture
- **Name:** Person's full name
- **Title:** Professional role/position
- **Bio:** Brief description
- **Stats:** Three metrics (projects, clients, experience)
- **Social Links:** Quick access to social profiles
- **Actions:** Follow and Message buttons

## How to Use

1. **Follow Button:** Click to toggle follow/unfollow state
2. **Message Button:** Click to open message dialog
3. **Social Links:** Click emoji buttons to visit social profiles
4. **Hover Effects:** Cards lift up with enhanced shadow on hover
